有点困有点晕

发表于 2017-11-02 | 分类于 随便想想

应该是最近知乎逛多了，脑子里总有莫名奇妙的想法。最近不在状态，Rick and Morty 看多了，觉得自己分化了。。一部分(Reality)渴望找到好工作，努力攒钱生活，另一部分(Spirit)变的哲学化，思考人生，过于物质真的是我们想要的吗？？现在有点困，敲敲文字瞎想想:)

多少人挤进IT行业只是为了混口饭吃，为了可观的工资去首付一套房子。但是，真正的科技和财富都集中在顶尖的公司和个人手中，想改变整个世界或者是简单点(也不简单)，想达到财富自由，这两方面，都得不停不停地奋斗。

一个普通的码农：总在逛房产网，二手车网，在考虑月薪，考虑和金钱有关的一切。。虽然我还有一段时间才去找实习，但是现在就患上了这种“普码症”。加过一个WH本地的IT群，还碰到几个老乡，群里感觉就有阶级分层，BAT的，国外的，本地有名的，本地外包的-.-。

钱不是万能的，没钱却是万万不能的。互联网的存在造就了一大批富翁，也可以说是，这部分人的才智和干劲创造了财富。一个程序员为了一个月5K,10K,30K去敲代码不是坏事，但是也得想远点，即使我们攒够了首付，一年能拿一百万，和大部分奋斗的IT人一样，除了这些，还剩什么呢。

一个二线的WH就有1000+的千万富翁,一个北京有数不清的带房亿万富翁，社会的阶级分层越来越明显了，底层的孩子想出人头地必须得读读读读书，要有长远的规划，每一条路都要走对。

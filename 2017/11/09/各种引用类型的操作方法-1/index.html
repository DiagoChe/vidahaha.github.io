各种引用类型的操作方法-1

发表于 2017-11-09 | 分类于 web开发 ， api大全

概括

计划把es5,es6中的所有引用类型(Object, Array, Date, RegExp, Function, Boolean, Number, String, Global, Math)的各种操作方法给上手总结下，熟悉和理解。

Object

Object构造函数

Object构造函数为给定值创建一个对象包装器。如果给定值是null或undefined，将会创建并返回一个空对象，否则，将返回一个与给定值对应类型的对象。

Object.assign()

Object.assign(target, …sources) 方法用于将所有可枚举属性的值从一个或多个源对象复制到目标对象。它将返回目标对象。

Object.create()

Object.create(proto[, propertiesObject]) 方法会使用指定的原型对象及其属性去创建一个新的对象。

Object.defineProperties()

Object.defineProperties(obj, props) 方法直接在一个对象上定义新的属性或修改现有属性，并返回该对象。

Object.entries()

Object.entries(obj) 方法返回一个给定对象自身可枚举属性的键值对数组，其排列与使用 for…in 循环遍历该对象时返回的顺序一致（区别在于 for-in 循环也枚举原型链中的属性）。
